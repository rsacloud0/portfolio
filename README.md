# Portfolio

My personal portfolio website built with Next.js.

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.

## Stack

- Next.js 16
- TypeScript
- Tailwind CSS v4
- Framer Motion
- MDX (blog content)
